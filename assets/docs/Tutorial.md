# 使用教程

## 运行前准备

| 平台 | 本地游戏 | 云游戏 |
|------|---------|--------|
| Windows | ✅ | ✅ |
| macOS | ❌ | ✅ |
| Linux | ❌ | ✅ |
| Docker | ❌ | ✅ |

本地游戏模式仅支持 Windows，必须使用 PC 客户端，以 1920×1080 分辨率窗口运行游戏；如果窗口模式不可用，再选择全屏。

游戏内语言必须设置为简体中文，并关闭帧率监控 HUD、微星小飞机、游戏加加、HDR、N 卡游戏滤镜等可能遮挡或改变画面的软件。

云游戏模式支持 Windows、macOS、Linux。macOS / Linux 用户可通过源码运行使用云游戏模式，也可以使用 [Docker 部署](/assets/docs/Docker.md)。

> 开启 `云·星穹铁道` 后，清体力、每日实训、差分宇宙等任务不再要求固定窗口，
>
> 支持后台运行；锄大地和模拟宇宙仍需要保持前台全屏。
>
> 如果有后台运行或多开桌面的需求，也可以使用 [远程本地多用户桌面](/assets/docs/Background.md)。

## 下载安装

点击 [Releases](https://github.com/moesnow/March7thAssistant/releases/latest) 打开下载页面，滚动到底部。

下载名称类似 `March7thAssistant_full.zip` 或 `March7thAssistant_full.7z` 的完整包。

> March7thAssistant 接入了第三方服务 [Mirror酱](https://mirrorchyan.com/?source=m7a-tutorial)。
>
> 如果中国大陆下载缓慢，可以尝试 [点我前往 Mirror酱 高速下载](https://mirrorchyan.com/zh/download?rid=March7thAssistant&os=&arch=&channel=stable&source=m7a-tutorial)。

下载完成后，右键选择 “全部解压缩” 或使用 [第三方压缩软件](https://yasuo.360.cn/) 解压。

解压后的目录中通常至少包含以下文件（夹）：

- `March7th Launcher.exe`
- `March7th Assistant.exe`
- `March7th Updater.exe`
- `assets`
- `libraries`
- `3rdparty`

双击 `March7th Launcher.exe` 打开图形界面，并同意《免责声明》。

如果你需要配合系统任务计划程序定时运行，或希望直接执行完整运行，也可以使用 `March7th Assistant.exe`。

### 命令行启动

`March7th Launcher.exe` 支持命令行参数，可以在启动图形界面的同时自动开始指定任务。

通过终端运行时，请使用 `管理员模式` 打开的 PowerShell、Windows Terminal 或 CMD；

如果你使用的是支持 `Sudo for Windows` 的系统，也可以用 `sudo` 来执行命令。

根据微软文档，`Sudo for Windows` 仅在 Windows 11 版本 24H2 或更高版本中可用，

使用前还需要先在系统设置中启用。详情请参考：[Sudo for Windows](https://learn.microsoft.com/zh-cn/windows/advanced-settings/sudo/)

常用示例：

March7th Launcher.exe -h （查看帮助）

March7th Launcher.exe daily （启动图形界面并执行每日实训）

March7th Launcher.exe main -e （任务正常完成后自动退出程序）

常见任务名称包括：`main`（完整运行）、`daily`（每日实训）、`power`（清体力）、`currencywars`（货币战争）等。

如果你只记得要运行某个任务，但忘了参数名，直接执行 `March7th Launcher.exe -l` 查看最新列表即可。

## 首页任务

### 完整运行

`完整运行` 是最推荐的入口，适合日常无人值守。

程序会根据你当前的设置，只执行 `已启用` 且 `已刷新` 的内容。当前版本中，完整运行通常会串行处理以下内容：

兑换码领取、每日实训、活动、历战余响、清体力、货币战争积分奖励、差分宇宙积分奖励、

锄大地、模拟宇宙 / 差分宇宙、混沌回忆、虚构叙事、末日幻影，以及最后的领取奖励。

没有开启的任务会自动跳过；已经判断为完成、尚未刷新到下一周期的任务也不会重复执行。

默认刷新时间为每天凌晨 4 点；周常内容一般在每周一凌晨 4 点刷新。

### 日常

首页 `日常` 菜单包含 `每日实训` 和 `清体力`。

`每日实训` 会先识别尚未完成的实训任务，再尝试自动完成；如果你只想单独刷本耗体力，直接使用 `清体力` 即可。

`清体力` 会根据你在设置中选择的副本类型和副本名称自动计算挑战次数，

也会配合后备开拓力、燃料、支援角色、培养目标等选项一起生效。

### 货币战争

当前版本已经支持 `货币战争·零和博弈`。首页入口提供 `运行一次` 和 `循环运行` 两种方式。

如果你在设置里开启了 `货币战争积分奖励`，完整运行会按周常逻辑检查积分奖励是否已经完成；

如果没有完成，就会自动进入货币战争处理本周积分。

设置页中可以进一步配置 `标准博弈 / 超频博弈`、快捷配置、职级难度、策略、速通模式，

以及领取积分奖励后是否自动执行位面饰品快速提取。

### 差分宇宙 / 模拟宇宙

当前版本已经支持 `差分宇宙·乐园漫记`。首页入口提供 `运行一次` 、 `循环运行` 和 `中途接管` 三种方式。

如果你在设置里开启了 `差分宇宙积分奖励`，完整运行会按周常逻辑检查积分奖励是否已经完成；

如果没有完成，就会自动进入差分宇宙处理本周积分。

设置页中可以进一步配置 `常规演算 / 周期演算`、难度等级、站点优先级、低性能兼容模式，以及领取积分奖励后是否自动执行饰品提取。

> `启用模拟宇宙 / 差分宇宙` 配置页通常用于反复刷取遗器经验和灵之珠泪（代替敌方掉落素材）直到每周上限（100只精英怪）。
> 
> 其中模拟宇宙部分调用的是 [Auto_Simulated_Universe](https://github.com/CHNZYX/Auto_Simulated_Universe) 项目，目前处于维护停滞的状态，如遇到问题建议改用差分宇宙。

### 锄大地

锄大地调用的是 [Fhoe-Rail](https://github.com/linruowuyin/Fhoe-Rail) 项目。首页菜单提供 `快速启动`、`原版运行`、`更新锄大地`、`重置配置文件`、`打开程序目录`、`打开项目主页`。

如果需要从中断的地图继续运行，可以使用 `原版运行` 打开锄大地自身界面，再在原项目里选择对应模式继续。

使用前请先开完需要的锚点，并清掉会干扰传送或寻路的任务。

成年女性远程无位移模型通常更稳定，例如艾丝妲、三月七；黄泉等角色可以配合专用地图版本使用。

> 帧数要求 60 帧，其余画质设置一般影响不大。
> 
> 锄大地跑图路线均为人工公益录制，若您有兴趣的话欢迎上传一些优化后的路线/新版本地图路线，减轻一下维护人员的负担~

### 逐光捡金

当前版本中的 `逐光捡金` 分为三类挑战：

- `混沌回忆`
- `虚构叙事`
- `末日幻影`

这三项都可以单独启用，并分别配置关卡范围和两队阵容。完整运行会在周常刷新后按你的开关与配置依次处理；

如果只想单独打其中一种，也可以直接从首页进入对应任务。

### 工具箱

左侧导航栏的 `工具箱` 提供一些常用辅助功能：

- `游戏截图`：检查程序是否正确获取游戏画面，也支持框选 OCR 识别文字。
- `自动对话`：进入剧情界面后自动开始运行，不支持云·星穹铁道。
- `兑换码`：执行当前配置中的兑换码。
- `解锁帧率`：在支持的平台上切换 60 / 120 帧。
- `触屏模式`：按云游戏移动端 UI 方式启动游戏，依赖 [第三方项目](https://github.com/winTEuser/Genshin_StarRail_fps_unlocker) 实现。

如果你遇到画面识别异常，最先应该打开 `工具箱 → 游戏截图` 检查采集到的画面是否正确。

## 流程编排

左侧导航栏中的 `流程编排` 适合制作自定义自动化流程。

它支持把图片点击、文字点击、坐标点击、等待、按键、界面切换、消息推送等步骤拼接起来，

也支持 `if`、`for`、`while`、`break`、`continue` 这类控制逻辑。

如果你的需求已经能被首页现成任务覆盖，优先使用首页任务即可；

如果你需要补充自己的个性化逻辑，例如某个固定界面的重复点击、简单分支判断或循环检测，再使用流程编排会更合适。

首次使用建议先从一个很短的小流程开始，例如：识别按钮、点击按钮、等待 1 秒；

调试时优先使用“执行选中步骤”，确认单个节点稳定后，再逐步扩展完整流程。

详细说明请查看 [流程编排使用教程](/assets/docs/Workflow.md)。

## 配置说明

点击左下角的 `设置` 进入小助手设置界面。

### 程序设置

#### 日志等级

一般保持默认 `简洁` 即可；如果遇到异常，请切换为 `详细` 方便排查问题。

#### 游戏路径 / 启动器路径

启动过游戏后再打开小助手，通常会自动配置 `游戏路径`。如果手动选择，

注意不要误选启动器 `miHoYo Launcher\launcher.exe`，真正的游戏本体是 `Star Rail Game\StarRail.exe`。

如果你希望通过米哈游启动器更新或预下载游戏，还需要额外配置 `米哈游启动器路径`。

#### 任务完成后

当前版本支持 `无`、`退出`、`关机`、`睡眠`、`休眠`、`重启`、`注销`、`关闭显示器`、`运行脚本`、`循环` 等完成后操作。

其中 `退出` 指退出游戏；`运行脚本` 需要额外填写路径；`关机`、`睡眠`等操作会先提示并等待一段时间，方便你取消。

#### 定时运行

`循环模式` 仍然保留用于兼容旧配置，但新版更推荐直接在 `日志` 页面配置 `定时任务`。

定时任务支持：

- 启动内置任务，例如 `完整运行`、`清体力`、`货币战争`。
- 启动外部程序或脚本。
- 为单条任务设置超时强制停止。
- 配置完成后推送通知。
- 配置任务完成后的操作。
- 配置完成后自动终止指定进程。

如果你是 7×24 小时挂机用户，优先考虑 `日志页定时任务`，不要再依赖旧的循环模式。

### 体力与日常设置

#### 副本类型 / 副本名称

清体力时刷什么副本，由这两项共同决定。程序会根据开拓力自动计算次数；

如果检测到例如 `位面分裂` 等双倍活动，也会优先使用这里配置的副本去完成。

#### 使用后备开拓力 / 使用燃料

这两个选项用于清体力时额外消耗后备开拓力或燃料。单次存在上限。

#### 支援角色

开启 `启用使用支援角色` 后，程序会在需要时尝试使用支援；开启 `强制使用支援角色` 后，每次打本都会优先尝试使用支援角色。

#### 培养目标

开启 `培养目标` 后，程序会根据目标自动匹配行迹或遗器副本；如果识别不到培养目标，才会回退到你手动配置的默认副本。

#### 历战余响

开启后，完整运行会优先完成每周三次 `历战余响`，并支持配置从周几开始执行。

### 货币战争设置

如果你只想每周自动拿满货币战争积分奖励，开启 `启用「货币战争」积分奖励` 即可。

常用配置项包括：

- `类别`：标准博弈 / 超频博弈
- `快捷配置`
- `职级难度`
- `货币战争策略`
- `遇到特定词条时接受重开`
- `启用速通模式`

### 差分宇宙 / 模拟宇宙设置

如果你只想每周自动拿满差分宇宙积分奖励，开启 `启用「差分宇宙」积分奖励` 即可。

`差分宇宙` 页面主要负责每周积分奖励、开图鉴、提升等级，

包括常规演算 / 周期演算、难度、低性能兼容模式、奖励后自动饰品提取、站点优先级等。

`模拟宇宙 / 差分宇宙` 页面则负责刷取遗器经验和灵之珠泪直到周上限（100精英怪），配置项包括：

- `运行模式`：集成 / 源码（仅模拟宇宙生效）
- `类别`：差分宇宙 / 模拟宇宙
- `运行频率`：每周 / 每天
- `运行次数`
- `自动领取模拟宇宙沉浸奖励`
- `命途`（仅模拟宇宙生效）
- `难度`（仅模拟宇宙生效）
- `启用模拟宇宙 GPU 加速`（仅模拟宇宙生效）

如果你选择 `源码` 模式，需要保证本机 Python 环境可用。

### 锄大地设置

锄大地页除了启用开关外，还可以配置：

- `运行模式`：集成 / 源码
- `自动切换队伍`
- `地图版本`
- `优先星球`
- `购买秘技零食并合成零食`
- `购买代币与过期邮包`

如果你选择 `源码` 模式，需要保证本机 Python 环境可用。

### 逐光捡金设置

`混沌回忆`、`虚构叙事`、`末日幻影` 都支持分别配置：

- 是否启用
- 关卡范围
- 两队阵容
- 上次运行时间

建议先把队伍配置好，再用完整运行接管周常。

### 云·星穹铁道

开启 `使用“云·星穹铁道”` 后，小助手会改用浏览器控制云游戏来执行清体力等自动化任务。

常用配置包括：

- `全屏运行`
- `最大排队等待时间`
- `浏览器类型`
- `使用国内镜像下载浏览器和驱动`
- `保存浏览器数据`
- `浏览器画面缩放（DPI）`
- `启用无窗口模式（后台运行）`
- `未登录时自动切换为有窗口模式`

其中 `无窗口模式` 不支持模拟宇宙（不包含差分宇宙）和锄大地；如果你需要后台挂机日常和清体力，这一组配置非常关键。

### 杂项设置

`杂项` 页主要是一些增强稳定性和自动化体验的辅助开关，常用的有：

- `启用自动战斗和二倍速检测`：在游戏启动前尝试开启自动战斗和二倍速，并在清体力、货币战争等内置战斗场景中维持状态。
- `启用自动修改分辨率并关闭自动 HDR`：通过小助手启动游戏时自动切到 1920×1080 并关闭自动 HDR，适合本地模式用户。
- `启用自动配置游戏路径`：会尝试从快捷方式、官方启动器或正在运行的游戏进程中自动识别路径。
- `优先使用后台截图`：默认建议开启，可以减少悬浮窗、覆盖层等对截图识别的干扰。
- `在用户登录时启动`：通过任务计划程序在登录后自动启动；如果想完全无人值守，通常还需要你自己配置系统自动登录。

关于 `OCR 加速模式`：建议先保持 `自动`。各模式说明如下：

- `自动`（默认）：Windows 10 18362 及以上优先使用 ONNXRuntime(DirectML) GPU 加速；
- 否则若检测到 OpenVINO 则使用 OpenVINO CPU；都不满足则回退到 ONNXRuntime CPU。
- `GPU (ONNXRuntime DirectML)`：强制使用 DirectML GPU 加速，环境不支持时自动回退。
- `CPU (OpenVINO)`：强制使用 OpenVINO CPU 推理。OpenVINO 在纯 CPU 场景下通常比 ONNXRuntime CPU 更快，
- 但**存在内存持续增长的问题**——程序会定期重新初始化引擎以释放内存，并在可用内存不足时降级到 ONNXRuntime CPU。
- `CPU (ONNXRuntime)`：强制使用 ONNXRuntime CPU 推理，兼容性最好、内存行为最稳定。
- 如果遇到识别异常、闪退或内存占用过高，可手动切换为 `CPU (ONNXRuntime)`。

### 按键设置

按键设置位于 `杂项` 页，只对小助手内置功能生效，可用于配置秘技、地图、跃迁、停止任务等快捷键。

其中 `秘技` 按键主要作用于清体力和逐光捡金等内置任务。

如果你需要修改模拟宇宙或锄大地的按键，请查看对应第三方项目自己的文档和配置文件。

### 账号管理（仅 Windows）

`账号` 页只在 Windows 下显示，主要用于本地游戏模式的多账号导出与切换，不支持 `云·星穹铁道`。

使用时建议按这个顺序：

1. 先完整进入游戏，直到看见角色站在大世界中，再进行导出。
2. 点击 `导出当前游戏账户`，把当前账号保存到本地列表。
3. 以后需要切换时，在列表中选中目标账号，点击 `导入选中的账户`。

账号页还支持这些操作：

- `账户更名`：给本地保存的账号改一个更容易识别的名字。
- `删除账户`：删除本地保存的账号数据。
- `刷新`：重新读取本地账号列表。
- `清除注册表`：清理当前机器上保留的相关账户信息，适合排查切号异常。

`自动登录` 会额外保存你为该账号输入的用户名和密码，并在登录过期时尝试自动回填。

这个功能有一定风险，只建议在你明确知道自己在做什么时使用。

> 重要：如果你在还没完全进入游戏时就导出账号，可能会读取到上一个账号的 UID，导致保存的文件和实际账号不对应。

## 更新

出现新版本弹窗后，点击下载即可；你也可以在设置页最底部点击 `检查更新`，或者手动双击 `March7th Updater.exe`。

如果你手动更新，请下载名称类似 `March7thAssistant.zip` 或 `March7thAssistant.7z` 的完整包，

解压后将全部文件覆盖到旧版本目录即可。
