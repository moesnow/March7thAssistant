它为我们提供了免费的检查更新接口，但它的下载是有偿的，需要用户付费使用。

不过，即使不购买 Mirror 酱的下载服务，你也可以在检测到更新后，在设置里选择从海外源（GitHub）下载~

如果你购买并填写了 CDK，更新时就再也不用和自己的网络环境斗智斗勇咯，下载更快更稳定！

同时 CDK 也可以用于其他接入 Mirror酱 的项目，例如 MAA 等。

### Q：小助手启动慢/未找到可执行文件/进程启动失败/总是被杀毒软件删除 怎么办？

A：请将`小助手文件夹`加入杀毒软件排除项/白名单/信任区，然后重新解压覆盖一次。

Windows Defender：`病毒和威胁防护` → `管理设置` → `添加或删除排除项` 可以参考 [#373](https://github.com/moesnow/March7thAssistant/issues/373) 图示操作

（注意：可能还需要将 `应用和浏览器控制` → `智能应用控制设置` → `切换为关闭` ）

火绒安全软件：`主界面` → `右上角菜单` → `信任区` 可以参考 [教程](https://cs.xunyou.com/html/282/15252.shtml) 图示操作

### Q：开了滤镜、帧率显示或性能检测软件后，为什么总是识别失败/提示切换到界面失败？

A：这类软件会在游戏画面上叠加额外文字或特效，导致小助手的图像识别和 OCR 判断失败。

常见影响项包括：`N 卡游戏滤镜`、`HDR`、`帧率监控 HUD`、`微星小飞机 / Afterburner`、`RTSS`、`游戏加加`，

以及各种会显示 `FPS / CPU / GPU` 的悬浮窗。

另外，部分笔记本/主板厂商自带软件也会提供类似的功能，例如 `护眼模式`、`鲜艳模式`、`游戏增强`、`色彩优化`、`场景模式` 等，

这些功能即使没有明显悬浮窗，也可能直接改变画面的颜色、亮度、对比度。

请先完全关闭这些滤镜、硬件监控和性能检测软件后再重试。

有些滤镜或画面增强因为你自己已经看习惯了，反而最不容易发现；

如果没有看到大量其他用户同时报同样的错误，通常优先按你自己电脑上的环境问题排查。

如果日志里出现了类似 `检测到疑似监控悬浮窗文字` 的提示，基本就是这个问题。

你也可以打开小助手 `工具箱 → 游戏截图`，检查程序实际获取到的画面是否干净、是否被悬浮窗遮挡。

### Q：差分宇宙/模拟宇宙/饰品提取怎么不会选择角色/自动选择 XXX 角色，如何更换角色？

A：以上功能使用游戏内自动记忆的配队，如果检测到缺少角色，会自动尝试选择 “队伍1”。

如需更换角色，可以手动配置好队伍后，进入一次对应的界面，随后退出检查游戏内记忆的配队是否更新即可。

### Q：为什么不会自动战斗/不会交互随意门/角色在乱跑？

A：这类问题大多是 `按键设置和小助手默认假设不一致` 导致的，可以按下面顺序排查：

1. 先检查你是否修改过游戏内默认按键，重点看：自动战斗（`V`）、交互（`F`）、角色移动（`WASD`）。如果改过，建议先恢复成默认键位再测试。

2. 再检查小助手 `设置 → 杂项 → 启用自动战斗和二倍速检测` 是否已经开启。

3. 如果问题只出在自动战斗，再检查游戏内 `设置 → 战斗功能 → 是否沿用自动战斗设置` 是否为“是”；必要时可以先手动开启一次自动战斗后再试。

### Q：开始运行后就不能移动键盘鼠标和切换到后台了吗

A：本地游戏模式下，大多数自动化仍然依赖当前桌面画面和输入状态，运行时不建议抢占键盘鼠标，也不建议切到后台。

如果你需要后台运行，优先考虑使用 `云·星穹铁道`：目前清体力、每日实训、货币战争、差分宇宙等任务支持后台运行；

锄大地和模拟宇宙仍需要前台全屏。

如果你需要本地模式后台挂机，也可以尝试 [远程本地多用户桌面](https://m7a.top/#/assets/docs/Background)。

### Q：完整运行的作用是什么

A：`完整运行` 会根据你当前的开关和刷新状态，自动处理已启用的内容，通常包括：

兑换码、每日实训、活动、历战余响、清体力、货币战争积分奖励、差分宇宙积分奖励、

锄大地、模拟宇宙 / 差分宇宙、混沌回忆、虚构叙事、末日幻影，以及最后的领取奖励。

已经完成或尚未刷新的内容会自动跳过。默认日常刷新时间为每天凌晨 4 点，周常刷新时间一般为每周一凌晨 4 点。

### Q：提示窗口找不到怎么办

A：请检查是否将游戏内语言设置为简体中文。

### Q：缺少新增加的副本？逐光捡金缺少新角色？

A：首先确保是最新版本，如果仍然缺少副本，编辑 `assets\config\instance_names.json` 文件手动添加。

如果 `混沌回忆 / 虚构叙事 / 末日幻影 / 支援` 中缺少新角色，

可以在对应界面里用小助手工具箱中的`捕获截图`功能，勾选角色头像，点击`保存所选截图`后，

放入 `assets\images\share\character` ，并修改 `assets\config\character_names.json` 文件。

同时欢迎用`未压缩`的方式上传到 [Issue](https://github.com/moesnow/March7thAssistant/issues)，比如压缩包，或者 [PR](https://github.com/moesnow/March7thAssistant/pulls) 也可以。

### Q：运行过程中内存占用持续升高 / 游戏突然无响应或闪退 / 系统崩溃重启怎么办？

A：如果出现内存占用异常偏高、程序无故闪退，或运行途中游戏画面出现异常，优先排查 OCR 加速模式的兼容性。

小助手默认在 Windows 10 18362 及以上使用 ONNXRuntime DirectML GPU 加速；

在不支持 DirectML 的环境（如 Linux、部分 ARM 设备）下，若检测到 OpenVINO 则自动使用 OpenVINO CPU 推理。

**OpenVINO 在某些机器上存在内存持续增长的问题**，程序会定期重建推理引擎来释放内存，

并在可用物理内存低于 4GB 时自动切换到 ONNXRuntime CPU；但如果机器本身内存较小，仍可能出现明显波动。

**建议先将 OCR 加速模式手动切换为 `CPU (ONNXRuntime)`**（设置 → 杂项 → OCR 加速模式），

切换后重新运行，观察内存和稳定性是否改善。如果问题消失，说明原模式与当前环境存在兼容问题，保持 ONNXRuntime CPU 即可。

如果使用 DirectML GPU 加速时出现闪退或识别异常，同样建议先回退到 `CPU (ONNXRuntime)` 排查。

### Q：双击启动后窗口一闪而过 / 图形界面根本打不开 / 终端里运行命令时窗口一闪而过怎么办？

A：这几种现象通常可以放在一起排查。

如果你是直接双击 `March7th Launcher.exe` 或其他可执行文件后，发现终端窗口一闪而过、图形界面根本没有打开，

建议改用终端手动启动，这样更容易看到具体报错信息。

请使用`管理员模式`打开 PowerShell、Windows Terminal 或 CMD 后，再进入小助手目录执行对应命令。

例如可以先尝试：

PowerShell 使用：

```powershell
& '.\March7th Launcher.exe' -S
```

CMD 使用：

```cmd
"March7th Launcher.exe" -S
```

如果你的系统支持 [Sudo for Windows](https://learn.microsoft.com/zh-cn/windows/advanced-settings/sudo/)，也可以直接使用 `sudo`。

根据微软文档，`Sudo for Windows` 仅在 Windows 11 版本 24H2 或更高版本中可用，使用前还需要先在系统设置中启用。

如果改用终端启动后能看到报错，请优先根据报错内容排查；

常见原因仍然是权限不足、杀毒软件误删文件、依赖被拦截，或者工作目录不正确。
